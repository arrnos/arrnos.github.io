Ubuntu 16.04 LTS安装好之后需要做的几件事

发表于 2016-10-20   |   分类于 编程   |     |   阅读次数

前言

Ubuntu16.04 系统安装成功后，还不适合我们工作和学习的需要，还需要进行一些必要的配置。

正文

1. 更新软件源

在系统设置中，点击软件和更新，在里面的来源中选择其他站点中的 中国 ，现在比较好的下载源是 清华 ，如下图：

2. 更换语言

有的时候，一些朋友对英文界面不太熟悉，因此设置为中文语言显得很有必要。 (chinese(simplied)可能在列表下方，需要向下翻看)

具体步骤参见：

http://jingyan.baidu.com/article/3c343ff7029de30d37796306.html

####3. 挂载硬盘
点击一个虚拟机，在开启之前，点击“开启此虚拟机”下面的“编辑虚拟机”，在弹出的选项设置中开启共享文件夹，添加需要挂载的文件夹即可。

4. 安装搜狗输入法

推荐搜狗 Linux 输入法下载安装，有两种安装方法：

  • 命令行方式：
    sudo apt-get update sudo apt-get install sogoupinyin
  • 可以去搜狗输入法官网下载 DEB版本，可以dpkg 安装或直接双击安装deb安装包，这里就不过多介绍了。 （下载地址）

安装成功后，进行输入法设置：

系统设置>语言支持>键盘输入方式系统， 然后选择 fcitx 项，
然后重启系统即可使用。

5. 点击图标最小化

Ubuntu 16.04 LTS 也支持了点击应用程序 Launcher 图标即可「最小化」的功能，不过还是需要用户进行手动启用。

方法有两种：

  • 你可以安装 Unity Tweak Tool 图形界面工具之后在 「Unity」-「Launcher」-「Minimise」中进行配置。

  • 直接在终端中使用如下命令启用。
    gsettings set org.compiz.unityshell:/org/compiz/profiles/unity/plugins/unityshell/ launcher-minimize-window true

    6. 安装系统清理工具

    操作系统使用久了，越来越多的缓存和垃圾文件都会不断拖慢系统，在此推荐大家安装开源、免费的 BleachBit 工具来进行 Ubuntu 系统清理。

    sudo apt-get install bleachbit

    bleachbit 下载地址
    根据自己的版本下载，deb包，然后直接双击即可。

    7. 安装vim

    vim申请系统中没有集成，只能我们自己安装。

    sudo apt-get install vim

    8. 安装WPS Office

    目前MS一直不出Linux版的Office，只能凑合着用WPS了

    sudo apt-get install wps-office

    9. 删除libreoffice（根据自己实际情况）

    libreoffice虽然是开源的，但是Java写出来的office执行效率实在不敢恭维，装完系统后果断删掉

    sudo apt-get remove libreoffice-common

    10.删掉基本不用的自带软件（不要都删。。）

    sudo apt-get remove thunderbird totem rhythmbox empathy brasero simple-scan gnome-mahjongg aisleriot gnome-mines cheese transmission-common gnome-orca webbrowser-app gnome-sudoku landscape-client-ui-install

    sudo apt-get remove onboard deja-dup

这样系统就基本上干净了

附录：Ubuntu最常用命令

  1. ls 查看指定目录下的内容
    ls -l 查看当前目录内容的详细信息。
    ls -a 查看当前目录中的所有文件(包括隐藏文件)
  2. mkdir 创建单个目录
    mv 重命名
  3. cd .. 上一层目录
    cd ~ 主目录
    cd ./balba/ 进入此目录下的子目录
    cd /opt/java/ 进入绝对目录
  4. rmdir balabala 用来删除空目录
    touch 创建一个空文件,如touch foo.c
    rm 删除文件 如 rm aaa.c
    cp filename newfilename.c 复制文件
  5. 将a目录用gzip打包，tar zcvf a.tar.gz a
    将a目录用bzip2打包，tar jcvf a.tar.bz2 a
    解压gzip包，tar zxvf a.tar.gz
    解压bzip2包，tar jxvf a.tar.bz2
  6. find命令 在目录中查找文件 如 $find . -name ‘chess.*‘

  7. grep 在指定文件中搜索特定内容，并将含有这些内容的行输出到标准输出。若不指定文件名，则从标准输入读取。 如
    $grep ‘printf’ ./ -R./chess.c:30: printf(“\n “);
    ./chess.c:32: printf(“ %d”, i + 1);
    ./chess.c:33: printf(“\n”);
    ./chess.c:35: printf(“ -“);
    ./chess.c:36: printf(“\n”);

  8. ifconfig 查看网络设置

后记

具体设置请根据个人工作需求和喜欢进行详细设置。比如删除Amazon的链接，设置时间使用UTC，安装Chrome，安装Sublime Text 3，java，spark，经典菜单指示器，安装git和vpnc，openssh-server等等，详见参考链接2.

重装系统后，如何恢复呢：
需要安装github for Windows ，登录；
下载安装node.js，重开bash窗口，使用npm命令安装hexo，安装依赖包：$ npm install hexo-deployer-git –save，然后便可以使用hexo -s-g -d 命令了
