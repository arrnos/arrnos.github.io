Ubuntu环境下IDEA及Scala插件的安装及程序在spark上的运行

发表于 2016-10-27   |   分类于 编程   |     |   阅读次数

前言

在Ubuntu上将spark环境配置好之后，就需要进行程序的开发了。
我们一般在IDEA上开发Scala或java程序，下面重点讲解Ubuntu环境下IDEA的安装、Scala插件的安装、Scala项目的建立以及如何在spark集群上运行程序。

正文

一. Ubuntu环境下IDEA的安装

1. 下载idea

去官网https://www.jetbrains.com/idea/download/ 下载idea的Linux版本

2. 将下载的tar.gz包解压提取到你本地的安装目录下，我的是/opt/

3. 找到idea安装目录下的./bin/idea.sh ,在终端打开它，即可看到经典的idea启动界面


4. 启动idea之后，可以导入配置，或重新配置你的idea，然后新建项目或导入原来的项目

二. Scala插件的安装

1. 启动IDEA，在初始界面上，点击Configure>Plugins,然后在弹出的界面中，点击安装插件install ..plugins。 在插件安装界面上方，输入Scala 回车，然后右面有显示Scala插件安装按钮，点击即可（我的已经安装所以没有安装提示了）


2. 一般情况下，由于软件包资源链接的原因，下载插件很慢，我们一般采取第二种方式，即在Plugins安装界面下方，点击最后一个：从磁盘安装插件。 然后浏览本地插件点击安装即可，非常迅速。 IDEA的Scala插件包可以在 网址 下载。或百度搜索输入 Scala plugin for idea，注意下载时注意版本的匹配哈！（不懂的童鞋再留言）

三. Scala项目的建立

安装好Scala 插件后，就可以建立Scala项目了，下面是其过程。

1. 在IDEA初始界面，新建项目，选择Scala，简单的代码可以直接选择Scala建立

2. 在弹出来的界面中，填入项目名称、存放路径、JAVA-HOME路径、SCALA-HOME路径，点击finish

3. 在新建Scala项目，在src目录下，新建Scala类，选择object类型，编写helloworld测试。

四. 在spark集群上运行程序

Scala程序如果要在spark平台上运行，还需要在项目中导入spark包库。

1. 在file> project structure > libraries 中，点击+号，选择java，然后找到spark的安装目录，添加lib目录下的 spark-assembly-1.6.2-hadoop2.6.0库。添加完成后，回到项目文件目录可以看到，spark库已经被成功添加。



2. 然后在import 包库，创建spark环境和变量，编写程序即可
